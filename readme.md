## Pen-Drive do Bolsonaro: Uma Obra de Ficção e Humor

GitHub Pages disponível em: [https://leo150250.github.io/pendrivedobolsonaro/](https://leo150250.github.io/pendrivedobolsonaro/)

Bem-vindo ao projeto mais improvável da internet: o Pen-Drive do Bolsonaro! Aqui você encontrará uma mistura de sátira política, referências intergalácticas, memes, receitas absurdas e arquivos que só poderiam existir num universo paralelo (ou num gabinete paralelo).

### Sobre o Projeto

Esta obra é totalmente fictícia e humorística, baseada (porcamente) em fatos reais, boatos, teorias da conspiração e memes da política brasileira. Qualquer semelhança com a realidade é mera coincidência, ou talvez não...

O projeto simula um pen-drive recheado de arquivos suspeitos, pastas secretas, receitas do STF, planos do centrão, selfies do Queiroz, músicas do centrão, doramas coreanos, e até um manual do orçamento secreto. Tudo isso apresentado numa interface web inspirada no Windows XP, com direito a sons de USB e barra de tarefas.

### Estrutura do Projeto

- **public/**
	- `conteudo.json`: O coração do pen-drive, com uma árvore de arquivos e pastas fictícias, misturando política, humor e ficção científica.
	- `index.html`: Interface principal, simulando o explorador de arquivos do Windows XP.
	- `script.js`: Lógica para navegação, sons, barra de tarefas e interação com o conteúdo.
	- `style.css`: Visual nostálgico, colorido e propositalmente brega.

Ainda, na raiz deste projeto, há alguns SVGs feitos com Inkscape, para caso queira mudar algumas imagens específicas da página.

### Destaques do Pen-Drive

- Arquivos como `Plano_Centrão_Dominação_Brasil.txt`, `Manual_do_Orcamento_Secreto.pdf`, `Receita_de_Pao_de_Queijo_do_STF.txt`, `virus_gabinete.exe`, `Selfie_do_Queiroz.jpg`, `Meme_Centrão_2026.png`.
- Pastas secretas, memes políticos, doramas, músicas e receitas de outro mundo.
- Links para sites fictícios e reais, incluindo o GitHub do projeto e o site do desenvolvedor.
- Mensagens de humor e alertas para não levar nada a sério!

### Como rodar

1. Abra o `index.html` em seu navegador.
2. Explore as pastas, clique nos arquivos, divirta-se com as mensagens.
3. Não compartilhe o pen-drive com ninguém (ou compartilhe, sei lá... Compartilharam até no Catar, não é?).

### Contribua!

Quer adicionar mais arquivos absurdos, memes ou teorias? Faça um pull request! O projeto é aberto para toda criatividade e zoeira. Basta editar o arquivo `conteudo.json` e incluir os seus próprios arquivos também!
